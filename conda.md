# conda

```
conda remove --name deepnude --all
```

执行以下命令来清理 Conda 缓存：

```
conda clean --all
```



### 清理包缓存

1. 清理包缓存：

   ```
   conda clean --packages
   ```

### 清理 tarballs

1. 清理 tarball 文件：

   ```
   conda clean --tarballs
   ```

### 清理日志文件

1. 清理日志文件：

   ```
   conda clean --logfiles
   ```
